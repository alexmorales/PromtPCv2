# Guía docente para el uso de prompts en el desarrollo del Pensamiento Computacional

## Objetivo

Fortalecer habilidades asociadas al pensamiento computacional mediante interacción guiada con chatbots avanzados.

## Competencias trabajadas

- Descomposición.
- Abstracción.
- Diseño algorítmico.
- Reconocimiento de patrones.
- Evaluación crítica.

## Metodología

### Fase 1. Explicación conceptual
El docente introduce el tema y explica las estructuras repetitivas.

### Fase 2. Construcción de prompts
Los estudiantes formulan prompts para resolver problemas.

### Fase 3. Análisis crítico
Se analizan errores y limitaciones de las respuestas generadas por IA.

### Fase 4. Refinamiento
Los estudiantes reformulan prompts para mejorar las soluciones.

### Fase 5. Reflexión metacognitiva
Se discute cómo la calidad del prompt influye en la calidad de la solución.

## Recomendaciones

- evitar dependencia automática de IA;
- promover análisis crítico;
- utilizar problemas contextualizados;
- validar las respuestas generadas.
