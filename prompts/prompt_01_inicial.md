# Prompt inicial

Te encuentras en una situación en la que necesitas contar la cantidad de veces que aparece una palabra específica en un texto largo que has escrito.

Diseña un algoritmo en PSeInt o Java utilizando estructuras repetitivas para resolver este problema.

El algoritmo debe:
- solicitar un texto al usuario;
- solicitar una palabra a buscar;
- recorrer el texto;
- mostrar cuántas veces aparece la palabra.

No es necesario validar mayúsculas o caracteres especiales.
