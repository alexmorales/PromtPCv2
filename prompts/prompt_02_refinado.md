# Prompt refinado

Diseña un algoritmo en Java y PSeInt que permita contar cuántas veces aparece una palabra dentro de un texto ingresado por el usuario.

Condiciones:
- utilizar estructuras repetitivas;
- utilizar funciones existentes del lenguaje;
- evitar funciones inexistentes;
- usar correctamente variables enteras y cadenas;
- mostrar claramente el resultado final;
- evitar errores de sintaxis;
- optimizar el recorrido del texto.

El algoritmo debe ser claro y fácil de comprender para estudiantes de programación inicial.
