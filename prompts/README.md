# Prompts utilizados en la estrategia

Esta carpeta contiene ejemplos de prompts diseñados para apoyar el desarrollo del Pensamiento Computacional (PC) en cursos iniciales de programación.

Los prompts fueron organizados en diferentes niveles:

- Prompt inicial.
- Prompt refinado.
- Prompt metacognitivo.
- Prompt orientado a depuración.
- Prompt orientado a análisis crítico.

El propósito no es únicamente obtener soluciones correctas, sino promover procesos de:
- descomposición,
- abstracción,
- análisis algorítmico,
- validación crítica,
- y reformulación de problemas.
