# Prompt metacognitivo

Analiza el siguiente algoritmo generado por IA para contar palabras dentro de un texto.

Identifica:
- posibles errores lógicos;
- problemas de eficiencia;
- errores de sintaxis;
- uso incorrecto de tipos de datos;
- posibles mejoras algorítmicas.

Explica por qué cada corrección mejora la solución propuesta.
